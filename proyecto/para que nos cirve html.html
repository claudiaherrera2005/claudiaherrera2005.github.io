¿Qué es el HTML?

Las siglas HTML vienen de “Hyper Text Markup Language” o Lenguaje Marcado de Hipertexto. Los códigos HTML son el lenguaje universal que se utiliza para crear y dar formato a los sitios web. Funcionan en cualquier sistema operativo (Windows, Mac, Linux, etc.) y con cualquier navegador (Chrome, Explorer o Mozilla). Aunque no se trata de un lenguaje de programación muy sofisticado, saber manejar HTML te permite insertar otros códigos más potentes, como los de Javascript, por ejemplo.
